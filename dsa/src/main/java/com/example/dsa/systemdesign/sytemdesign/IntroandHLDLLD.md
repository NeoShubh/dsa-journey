# System Design

> **Refer:** https://www.geeksforgeeks.org/system-design/getting-started-with-system-design/

System Design is the process of planning and structuring the architecture of a software system.

## HLD - High-Level System Design

HLD defines the overall architecture of a system and the main components interacting with each other.

It's a big-picture view of the system, services, and data flow.

It's useful for the Solution Architect.

---

## LLD - Low-Level System Design

LLD defines the internal implementation details of a system.

It's useful for the developers to have a blueprint of how modules, classes, and functions interact with each other.
